\doxysubsection{Detailed Description}
System responsible for handling inputs, sound and rendering for a scene from a \mbox{\hyperlink{classarcade_1_1IGame}{IGame}}. 

You are expected to create your own implementation of \mbox{\hyperlink{classarcade_1_1IGraphical}{IGraphical}} for instance SFML or NCurses class, inheriting from \mbox{\hyperlink{classarcade_1_1IGraphical}{IGraphical}} and implementing all of its methods.

This init method is called in order to initialize everything the graphical library needs to render a scene.

That can be a window, a texture / sprite cache, fonts, etc.

This method is called once at the beginning of the game, and subsequently if the game has been destroyed and needs to be restarted.

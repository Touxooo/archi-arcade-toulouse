\doxysubsection{Detailed Description}
This interface is used co create a game entity described by a vector of components, and a vector of tags. 

You are expected to implement a class inheriting from \mbox{\hyperlink{classarcade_1_1IEntity}{IEntity}}, and implementing all of its virtual methods.
